## “安卓界面展厅”应用程序
- 体验安卓用户界面设计语言的多样性，差异性
- 学习使用各式组件
- 最初的目的是帮助用户界面设计师了解谷歌材料设计语言，现在是实践酷炫的安卓用户界面组件。

- 开发规范
```
简单，线框图，素色，控制包体积
重交互
轻色彩
```
